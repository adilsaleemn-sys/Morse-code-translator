# 🔤 Morse Code Translator

A modern, lightweight, and easy-to-use **Morse Code Translator** that allows users to convert **text to Morse code** and **Morse code back to text** instantly. This project is designed for learning, education, and real-world experimentation with Morse communication systems.

---

## 📌 Overview

Morse Code is one of the oldest communication systems that uses a combination of **dots (.)** and **dashes (-)** to represent letters, numbers, and symbols.

This project brings that historical system into a **modern web interface**, making it easy for beginners, students, and developers to understand and use Morse code interactively.

---

## 🚀 Key Features

### 🔄 Bidirectional Conversion
- Convert **Text → Morse Code**
- Convert **Morse Code → Text**

### 🔊 Audio Output (Optional Feature)
- Dot and dash sound playback using Web Audio API
- Helps users learn Morse by hearing patterns

### 📋 Copy & Paste Support
- One-click copy output functionality
- Easy sharing of results

### ⚡ Real-Time Conversion
- Instant output while typing
- No page reload required

### 📱 Responsive Design
- Works on mobile, tablet, and desktop devices

### 🎯 Beginner Friendly
- Simple UI/UX
- Easy to understand logic

---

## 🧠 How It Works

The system is based on a **Morse Code Dictionary (Mapping Table)**:

1. Input text is captured from the user.
2. Each character is matched with its Morse equivalent.
3. The output is generated using a lookup object.
4. For reverse conversion, Morse code is split by spaces and decoded back to text.
